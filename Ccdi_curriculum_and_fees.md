### Billing Formula

```
Total Assessment = (Total Enrolled Units × ₱364.00)
                 + (Number of Lab Subjects × ₱1,656.00)
                 + ₱6,956.00 (fixed miscellaneous)
```

**Example — BSEET 1st Year, 1st Semester (full load, 2 lab subjects, 19 tuition units):**
```
Tuition  = 19 units × ₱364.00       = ₱6,916.00
Lab      = 2 subjects × ₱1,656.00   = ₱3,312.00
Misc     =                             ₱6,956.00
─────────────────────────────────────────────────
Total    =                            ₱17,184.00
```

## Notes for System Implementation

### Subject Seeder
When seeding the `subjects` table:
- Set `price_per_unit = 364.00` for all subjects (AY 2025-2026 rate)
- Set `lab_fee = 1656.00` for subjects where `has_lab = true`
- **Do NOT seed** PATHFIT or NSTP — these are non-tuition subjects and must not appear in fee assessments
- Subjects with units in parentheses `(2)` or `(3)` are non-tuition — either exclude or mark with a flag

### Irregular Students
An irregular student who takes subjects from both BSEET and BSEECT will be billed per subject:
- Each subject's tuition = `units × ₱364`
- Each lab subject adds `₱1,656`
- The fixed miscellaneous block of `₱6,956` is charged once regardless of mix

### Rate Update (AY 2024-2025 → AY 2025-2026)
If the rate changes again, update only `config/fees.php`:
```php
'tuition_per_unit'    => 364.00,   // update this
'lab_fee_per_subject' => 1656.00,  // update this
```
Then run `php artisan config:clear`. No code changes needed.
